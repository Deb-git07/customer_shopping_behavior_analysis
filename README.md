# customer_shopping_behavior_analysis
This report presents a comprehensive analysis of customer shopping behavior based on a dataset of 3,900 retail customers. The analysis was conducted using a full data pipeline: Python (Jupyter Notebook) for exploratory data analysis and preprocessing, MySQL for structured querying, and Power BI for interactive visualization and dashboarding.

📊 Customer Behavior Analysis
📌 Overview
End-to-end data analytics project analyzing customer shopping behavior using Python, SQL, and Power BI. The project focuses on extracting insights and presenting them through an interactive dashboard and report.

📂 Dataset
customer_shopping_behavior.csv
Contains customer demographics, purchase data, product categories, and subscription details.
